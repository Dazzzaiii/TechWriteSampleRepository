# TechWriteSampleRepository

How to Run the project

1. Navigate to the project folder
Open the folder where the repository was cloned.

2. Open the project
Locate the index.html file and:

3. Double-click it, or

4. Right-click → Open with → choose your preferred web browser

5. View the output
The web page will launch in your browser, showing the HTML/CSS content.

Notes:

No additional installations or dependencies are required.

For more information, check this README or explore the project files.
